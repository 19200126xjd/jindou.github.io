## 🔐 基于区块链的DDoS攻击情报共享系统（核心项目）

### 📌 项目背景
针对多组织协同防御场景中，攻击情报难以共享、数据易被篡改的问题，设计并实现基于联盟链的DDoS攻击情报共享系统。

### 🧠 技术方案
- 后端：SpringBoot + RESTful API  
- 区块链：FISCO BCOS + Solidity智能合约  
- 数据架构：链上存证 + 链下数据库（MySQL）  
- 安全机制：哈希校验 + 数据一致性验证  

### ⚙️ 核心功能
- 恶意IP上报与信誉评估机制  
- 黑名单动态管理与策略控制  
- 攻击情报链上存证与可追溯查询  
- 链上交易哈希与链下数据映射  

### 📊 实验与性能测试
- 多线程并发测试（10+线程）  
- TPS（每秒交易数）性能评估  
- 异常场景测试（数据篡改 / 非法访问）  

### 🎯 我的贡献
- 系统整体架构设计（链上+链下融合）  
- 智能合约设计（信誉值与黑名单管理）  
- REST API开发与测试方案设计  
- 完成系统功能测试、异常测试与一致性验证

## 📱 慧康管家APP（健康管理系统）

### 📌 项目简介
一款基于 Android 平台的个人健康管理应用，集健康数据记录、饮食分析、体重监测与运动管理于一体，帮助用户实现科学健康管理。

### 🧠 系统架构
系统整体划分为四大功能模块：

#### 🩺 1. 健康数据管理
- 用户注册与登录（保障数据安全与个性化）
- 健康档案管理（体重、饮食、运动等数据记录与查询）

#### 🍽 2. 饮食管理
- 食物热量查询（支持常见食物热量检索）
- 饮食记录（每日饮食数据录入）
- 饮食分析（基于记录数据提供饮食建议）

#### ⚖️ 3. 体重管理
- 体重变化记录与趋势跟踪
- BMI指数计算与健康状态评估

#### 🏃 4. 运动管理
- 运动记录（运动类型、时间、热量消耗）
- 日常运动数据统计

---

### ⚙️ 技术实现
- 开发语言：Java  
- 开发框架：Android  
- 数据存储：SQLite 本地数据库  
- 图片加载：Picasso  

---

### 🧪 功能实现亮点
- 多模块数据统一管理（饮食+运动+体重）  
- BMI自动计算与健康评估  
- 基于历史数据的饮食分析功能  
- 本地数据库实现数据持久化存储  

### 🎯 我的贡献
- 完成系统整体功能设计与模块划分  
- 实现健康数据、饮食、运动等核心模块开发  
- 设计SQLite数据库结构并实现数据存储  
- 完成界面交互逻辑与功能联调
